我们希望哈密量具有以下形式

可以使用matlab中kron函数，或qutip中qt.tensor函数构造出$a{+}$、$a$与$\sigma{z}$的扩展形式。进而直接表示出H。
通过matlab可以写出单光场下的哈密顿量。

%set hamitonian,and out put H,H0,H1
%w0:原子;w:光场
%assume h=1
%set up for adag a sz sp(+)and sm(-)
adag=upor(N);
a=lowor(N);
In=eye(N);
I2=eye(2);
sm = [0 1;0 0 ];%sigma-
sp = [0 0;1 0 ];%sigma+
sz = [-1 0;0 1];
%expand
sm1 = kron(In,sm);
sp1 =kron(In,sp);
sz1=kron(In,sz);
adag1=kron(adag,I2);
a1=kron(a,I2);
H0=w0*sz1+w*(adag1*a1);
H1=l*(adag1*sm1+a1*sp1);
end

时间演化

%设定参数
N=1;
w1 = 1.0  * 2 * pi;  % cavity frequency1
w0 = 1.0  * 2 * pi ; % atom frequency
g=[1 1]* pi;
%构造H,并特征值分解H
H = oneh(N, w0, w1, g);
[U,V]= eig(H); %U represent eigvector; V represent eigvalue
%set initial state
t=0;
phy_t0=zeros(2,1);
phy_t0(1)=1;
%save the wight of the initial state
wight_2=zeros(1000,1);
for n=1:1000
    t=t+0.0025;
    E_evolution=zeros(size(H));
    for j=1:2
        E_evolution(j,j)=exp(-1i*V(j,j)*t);
    end
    U_evolution=U*E_evolution/U;
    %U_evolution=U*E_evolution/U;%也可以，原因：U为幺正算符
    phy_t=U_evolution*phy_t0;
%     phy_wight= phy_t.* conj(phy_t);
    wight_2(n)=phy_t'*[-1 0;0 1]*phy_t;

%     hold on
end
% hold off
t=0.0025*(1:1000);
plot(t,wight_2,'*-')

function [ H ] = doubleh( N, w0, w1, w2, g)
    %其中：N ——总粒子数；w0 ——原子的振动频率；
    %w1 ——第一个光场的振动频率；w2 ——第二个光场的振动频率
    if class(g) == 'double' 
        g=g*eye(4);
    end
    %g 为耦合系数，可以是double 类型，也可以是行向量类型
    H0 =eye(2*N+1);
    H1 = eye(2*N+1);
    for i=1:N+1
        H0(i,i) = w1*(i-1)+w2*(N-i+1)-w0;
    end 
    for i=N+2:2*N+1
        H0(i,i) = w2*(2*N+1-i)+w1*(i-N-2)+w0;
    end

    for i=0:N-1
        H1( i+N+2, i+1 )=g(4)*sqrt(N-i);
    end
    for i=1:N
        H1( i+N+1, i+1 )=g(3)*sqrt(i);
    end
    for i=0:N-1
        H1( i+2, i+N+2 )=g(1)*sqrt(i+1);
    end
    for i=1:N
        H1( i, i+N+1 )=g(2)*sqrt(N-i+1);
    end
    H=H1+H0;
    end

function [ H ] = doubleh( N, w0, w1, w2, g)
    clear all

    N=3;

    w2 = 2.0  * 2 * pi;  % cavity frequency2
    w0 = 1.0  * 2 * pi ; % atom frequency
    g=[1 1 1 1]* pi;

    [U,V]= eig(H); 
    %U represent eigvector; V represent eigvalue
    t=0;

    phy_t0(1)=1;
    %save the wight of the initial state
    wight_2=zeros(100,1);

    %     plot(t,phy_wight(2),'*');
    %     hold on
        wight_2(n)=phy_t'*sz*phy_t;
    end
    % hold off
    t=0.01*(1:1000);
    plot(t,wight_2)

问题与后续工作

  1. 在计算哈密顿量的过程中，我们想当然的忽略了$(a^{\dagger}a+\frac{1}{2})$中的$\frac{1}{2}$。而这样的忽略并没有有利的证据，可能在计算一些其他的物理量的平均值时会产生不一样的结果。当然这不是一件麻烦事，可以在后续的工作中在程序中加上$\frac{1}{2}$这一项。并且在每个物理量计算时都使用两个程序来看一看结果会不会不一样。
  2. 对于一些参数诸如初始状态，光场与原子的振动频率，耦合系数等等的选取还需进一步探究。比如：若不同光场对原子的耦合系数不一样会有什么现象。
  3. 目前还处于最初阶段，只计算了原子能级的平均值，之后可以做找一些物理量例如光子个数等等来计算，给出更多的结果。
  4. 对多光场的计算，现在只写了2光场的。但粒子数守恒的方法是可以推广到任意光场的，之后可以写一个任意光场的哈密顿量构造的函数。
  5. 在时间演化过程的计算中，总粒子数是不随时间变化的，但关于这一点我还不是很清楚。如果不随时间变化，我希望找到物理上的解释，这就促使我去看量光和一些相关的论文；若在时间演化中N会有一定概率变化，那么就要考虑考虑如何这样的时间演化了。
